## {Tips} 성장하는 개발자의 직장 내 처세술

### _처세술_

- 일정은 항상 넉넉히 잡기
  - 1주 정도 걸릴 것 같으면 3주 정도로 잡기
- 자원의 20%는 항상 미래를 위해서 사용해야 함
  - ex) 하루 10시간 근무
  - 그중 2시간은 미래를 위해 써야 함
    - 2h; 공부, 독서, 인맥 관계, ...
